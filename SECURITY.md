# Security

BreakGlass stores operational reference data locally. Do not commit real credentials, keys, customer data, screenshots with sensitive values, database exports, or local prep/import folders.

## Reporting

For security issues, use a private channel to the repository owner or GitHub's private vulnerability reporting if it is enabled for the repository. Avoid posting sensitive exploit details, credentials, or private operational data in public issues.

## Local data

The app database is stored locally under the Tauri app data directory. Backups and exports may contain sensitive operational information and should be handled outside the repository.
