XXX Servers Out

★
network power + tag Updated 23m ago

Steps

  1. Verify power at rack PDU. Check UPS state (front panel green = OK).
  2. If UPS on battery: confirm grid is out on the wall outlet. Notify facilities.
  3. Console into DC01 via iDRAC. If unreachable, walk to rack.
  4. If servers powered but unreachable: check core switch ports for VLAN 20 uplink.
  5. Escalate to Sarah Chen (IGT) if gaming-side impact: 555-0123.
